Explain the inline , internal ,external css?

inline:- it contains the css property in the body section attached with element.This inline style is specified within the html tag using the style attribute

internal:- it contains the css property in the head section that is the css is internal within the html file

external:- it contains saperate css file which contains only style properties with the help of tag attribute.This css property written in a saperate file with (.css) extention and should be linked to the html document using link in head section

difference b/w flex row and column

flex row::- this row is the default property in flex direction.this row was used to place the elements in main axis

flex column:- this column is used to place the elements in cross axis

justify content allows to do what

it defines how the browser distribute space between and around content items along the main axis of a flex container and the inline axis of a grid container

difference b/w absolute and relative positioning?

Absolute:-absolute positioned elements are removed from the normal flow, and can overlap element
Relative:-relative positioned to its normal position.Setting the top,right,bottom and left properties

what is z-index

The z-index property specifies the stack order of an element
An element with greater stack order is always in front of an element with a lower stack order.
